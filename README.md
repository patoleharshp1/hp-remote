# 📱 HP Remote - Harsh 7 OS
This is the official web-controller for the **Harsh 7 DIY Phone** project.

## 🚀 Features
- **PWA Ready:** Install as an app on Android/iOS.
- **Dual-Mode UI:** Automatically hides download links when installed.
- **Gemini Integration:** Control AI prompts via the web interface.
- **Themed:** High-contrast Red/Black "Harsh Edition" design.

## 🛠️ Hardware Setup
- **Brain:** ESP32 (Master) + Arduino Nano (Display/Pen)
- **Screen:** 1.8" TFT LCD
- **Storage:** 16GB SD Card
